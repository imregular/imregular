## 💼 Featured Projects

### 🛡️ SafeRoute AI
A safety-first intelligent navigation system for India that recommends the **safest route** — not just the fastest — by analyzing street lighting, CCTV coverage, historical crime data, and proximity to safe havens like police stations and hospitals.
- **Tech Stack:** React, Node.js, PostgreSQL, Prisma, Google Maps API, OpenAI API, Groq, Sarvam AI
- **Highlights:**
  - 🧠 LLM-powered routing engine (GPT-4o-mini + LLaMA fallback via Groq) returns structured safety scores and route reasoning
  - 🗣️ Real-time multilingual voice navigation using Groq + Sarvam AI's Bulbul v3 TTS — supports multiple indigenous Indian languages
  - 📍 Consensual GPS tracking & SOS alert system with live location snapshots and emergency contacts
  - 🚨 Crowdsourced community safety reports weighted algorithmically to penalize dangerous routes in real-time

### ☁️ Smart AI Image Storage
An intelligent cloud storage solution powered by AI for smart image organization, captioning, and semantic search.
- **Tech Stack:** React, Node.js, Python, PyTorch, Hugging Face Transformers, Pinecone, PostgreSQL
- **Highlights:** AI-powered captioning, 768-dim vector embeddings, semantic search, microservices architecture

### 🤖 [ANN Churn Prediction App](https://github.com/imregular/ANN_churn_prediction_App)
Customer churn prediction using Deep Learning with TensorFlow and Keras. This model predicts whether customers will exit based on various features, helping businesses retain valuable customers.
- **Tech Stack:** Python, TensorFlow, Keras, Pandas, NumPy
- **Highlights:** Deep Neural Networks, Data Preprocessing, Model Deployment

### 🔗 URL Shortener
A fast and reliable URL shortening service with analytics and custom short links.
- **Tech Stack:** Node.js, React, Prisma, PostgreSQL, Tailwind CSS
- **Highlights:** Custom URLs, Click Analytics, Deployed on Railway + Vercel

### ✍️ [Let Me Blog](https://github.com/imregular/Let_Me_blog)
A modern blogging platform built with TypeScript, enabling users to create, publish, and share their stories with a beautiful, responsive interface.
- **Tech Stack:** TypeScript, React, Node.js
- **Highlights:** Authentication, Rich Text Editor, Responsive Design

### 💬 [YouChat](https://github.com/imregular/YOUCHAT)

- **Highlights:** Real-time Messaging, User Authentication
